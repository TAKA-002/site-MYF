======================================
node_modulesディレクトリについて
======================================

node_modulesディレクトリは、webpackのパッケージが入っている。
中身はよくわからないが、様々な機能を使うためのファイルの集合体になっている。

これをすべてgitで管理すると膨大な量のファイルを保存することになる。
よくないので、gitでの管理はしない。

よって、gitからクローンしてきたら以下のコマンドを実行してnode_modulesを復活させる必要がある


▼コマンド①
npm view webpack

これで使っているwebpackのバージョンを確認する


▼コマンド②
npm install --save-dev webpack@5.28.0

これでnode_modulesディレクトリが復活する
